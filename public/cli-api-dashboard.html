🔧 CLI API 调用仪表板

📋 说明

由于CloudBase HTTP触发器配置限制，我们使用CLI直接调用函数。请复制以下命令到终端执行：

📋 健康检查

cloudbase functions:invoke api

💰 成本统计

cloudbase functions:invoke api

🤖 AI聊天

cloudbase functions:invoke api

📚 招生建议

cloudbase functions:invoke api

🔍 函数状态

cloudbase functions:detail api

📝 自动化脚本

您也可以使用以下脚本来自动化调用：

# 创建测试脚本 echo 'cloudbase functions:invoke api' > test-api.sh chmod +x test-api.sh ./test-api.sh
