RAG组件库演示

可复用的RAG系统UI组件

TOC目录组件

功能说明

自动生成页面目录，支持滚动跟踪和快速导航。

使用方法

// 创建TOC组件 const sections = [ { id: 'section-upload', title: '文档上传' }, { id: 'section-search', title: '智能搜索' }, { id: 'section-documents', title: '文档管理' } ]; const tocId = ragUI.createTOC(container, sections);

演示效果

右侧会显示当前页面的目录导航。

文档上传组件

功能说明

支持拖拽上传、进度显示、文件预览等功能。

使用方法

// 创建上传组件 const uploaderId = ragUI.createUploader(container, { onSuccess: function(result) { console.log('上传成功:', result); } });

演示效果

智能搜索组件

功能说明

支持关键词搜索、相关度排序、结果高亮等功能。

使用方法

// 创建搜索组件 const searchId = ragUI.createSearch(container, { onResults: function(results) { console.log('搜索结果:', results); } });

演示效果

文档管理组件

功能说明

支持文档列表、查看、删除、统计等功能。

使用方法

// 创建文档管理组件 const managerId = ragUI.createDocumentManager(container, { onDocumentUpdate: function() { console.log('文档已更新'); } });

演示效果

主题切换

功能说明

支持多种主题样式，可动态切换。

使用方法

完整示例

功能说明

展示如何组合使用所有组件创建完整的RAG管理系统。

使用方法

// 完整系统初始化 const ragUI = new RAGUI({ apiBase: '/api/rag', token: localStorage.getItem('rag_token'), theme: 'default' }); // 创建TOC const sections = [ { id: 'section-upload', title: '文档上传' }, { id: 'section-search', title: '智能搜索' }, { id: 'section-documents', title: '文档管理' } ]; ragUI.createTOC(document.body, sections); // 创建各个组件 ragUI.createUploader(uploadContainer, { onSuccess: handleUpload }); ragUI.createSearch(searchContainer, { onResults: handleResults }); ragUI.createDocumentManager(docsContainer, { onUpdate: handleUpdate });

演示效果
