### 🔍 故障排除

1. **数据库连接问题**:
   - 确保 `meeting_data.db` 文件存在
   - 检查文件权限

2. **端口占用**:
   - 如果5000端口被占用，修改 `dashboard_sqlite.py` 中的端口号

3. **依赖问题**:
   - 运行 `pip install -r requirements.txt` 安装依赖

### 📞 技术支持

如有问题，请检查：
1. Python环境是否正确
2. 依赖包是否完整安装
3. 数据库文件是否存在
4. 端口是否被占用
